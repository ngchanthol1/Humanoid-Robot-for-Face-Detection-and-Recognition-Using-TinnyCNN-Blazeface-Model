First-Time Application Run

---------------------------------------------------------------------

1. Copy the image dataset and paste it into the following folder: robot\_project/dataset/mrdavid
2. Run blazeface\_detector.py to enable the face detection module.
3. Run train\_model.py to generate the final optimized .tflite model.
4. Run TinnyCNN\_FaceRecognition.py to start the face detection and recognition system.



Subsequent Runs

--------------------------------------------------------------------------------------

* Since the model is already trained and downloaded, for subsequent runs you only need to execute TinnyCNN\_FaceRecognition.py.
* The system will start much faster, taking less than 4 minutes to launch.
